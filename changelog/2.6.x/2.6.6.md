# 2.6.6 — 2026-06-11

## Added

- **`scripts/clean-mcpb.ts`** — post-pack bundle cleaner, synced from framework 0.10.6 ([#230](https://github.com/cyanheads/mcp-ts-core/issues/230)). Strips `node_modules/**` agent-doc entries (`skills/`, `.claude/`, `.agents/` trees and `SKILL.md` files) that root-anchored `.mcpbignore` patterns cannot reach.
- **Truncation enrichment on `clinicaltrials_get_field_definitions`** — `ctx.enrich.truncated({ shown, cap })` now fires in search mode, adding `truncated`, `shown`, and `cap` enrichment fields plus an automatic `notice` when results are capped by the `limit` parameter.
- **`techniques` skill** synced from framework 0.10.0 — catalog of reusable response/data-shaping patterns (outline-on-overflow reference).

## Changed

- **`createApp()` identity pair** (`name: 'clinicaltrialsgov-mcp-server'`, `title: 'clinicaltrialsgov-mcp-server'`) added to `src/index.ts` per framework 0.10.6 identity adoption requirement ([#231](https://github.com/cyanheads/mcp-ts-core/issues/231)). `lint:packaging` check 9 validates this pair.
- **`bundle` script** now appends `&& bun run scripts/clean-mcpb.ts dist/clinicaltrialsgov-mcp-server.mcpb` after `mcpb pack`, so agent-doc entries are pruned automatically on every bundle build ([#230](https://github.com/cyanheads/mcp-ts-core/issues/230)).
- **`.mcpbignore`** — patterns re-anchored to root (`/.claude/`, `/.agents/`, `/skills/`) per framework 0.10.1 ([#207](https://github.com/cyanheads/mcp-ts-core/issues/207)); prevents accidentally stripping nested `node_modules` runtime paths.
- **`lint-packaging.ts`** — updated to framework 0.10.6: new check 8 (post-bundle content guard for agent-doc entries) and check 9 (identity checks: `name`/`title` in `createApp()` and `manifest.json` `display_name` must match unscoped package name).
- **`check-framework-antipatterns.ts`** — picks up the `coerce-boolean-env-flag` antipattern rule from framework 0.10.0 ([#201](https://github.com/cyanheads/mcp-ts-core/issues/201)); scanner skips comment lines to avoid false positives.
- **Skills synced** to framework 0.10.x: `add-prompt`, `add-resource`, `add-service`, `add-test`, `add-tool`, `api-canvas`, `api-config`, `api-context`, `api-linter`, `api-mirror`, `api-testing`, `api-workers`, `code-simplifier`, `design-mcp-server`, `orchestrations`, `polish-docs-meta`, `release-and-publish`, `report-issue-framework`, `security-pass`.
