# 2.6.1 — 2026-06-02

## Added

- **`moreInfo` section** for `clinicaltrials_get_study_results` ([#64](https://github.com/cyanheads/clinicaltrialsgov-mcp-server/issues/64)). Covers `moreInfoModule` — limitations & caveats, certain-agreement disclosure restrictions, and the results point of contact. Follows the same summary/full split as the other four sections: full mode passes the raw module through; summary mode returns `hasLimitations`, `restrictionType`, and `pointOfContact`. `moreInfo` is included in the default `sections` (all), making `get_study_results` a complete substitute for the `resultsSection` that `get_study_record` no longer embeds since 2.6.0.

## Fixed

- **`docs/design.md`** corrected: documented the previously-undocumented `summary` param on `get_study_results`, fixed the stale NCT-ID cap (20, not 5), and added `moreInfo` to the sections list.
